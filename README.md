# 🎵 QR Music Player

An interactive web application that connects music, QR codes, and real-time visual experiences.

Users can scan a QR code assigned to a song, instantly play the audio, and enjoy dynamic visualizations that react to the music.

---

## 🚀 Overview

QR Music Player explores the intersection of web development, music, and interactive experiences.

The project aims to create a simple workflow:

1. Generate a QR code for a song.
2. Scan the QR code using a device camera.
3. Load and play the associated audio.
4. Display real-time visualizations synchronized with the music.

---

## ✨ Features

### QR Code Integration

* Generate QR codes for songs.
* Associate each QR code with a specific audio track.
* Scan QR codes directly from the browser.

### Audio Playback

* Play songs after scanning.
* Control playback state.
* Manage audio loading dynamically.

### Audio Visualization

* Real-time visual feedback.
* Canvas-based rendering.
* Audio-reactive animations powered by frequency analysis.

### User Experience

* Responsive interface.
* Simple and intuitive workflow.
* Mobile-friendly interaction.
